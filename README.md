# AI-Powered Interview Coach

An AI-powered interview preparation platform that personalizes mock interviews using resume analysis and job descriptions. The application automatically extracts candidate information from PDF resumes, generates role-specific interview questions, evaluates responses, and provides structured AI feedback to simulate a realistic technical interview experience.

---

## Features

- 📄 Upload and parse PDF resumes
- 🤖 AI-powered resume analysis using Gemini API
- 👤 Automatic candidate profile extraction
- 💼 Job description-based interview personalization
- 🎯 Dynamic interview question generation
- 📝 AI-based answer evaluation and scoring
- 💬 Contextual follow-up questions
- ⚡ REST API-based full-stack architecture

## Project Workflow

```

        ↓
Extract Resume Information
        ↓
Generate Candidate Profile
        ↓
Enter Company, Role & Job Description
        ↓
Generate Personalized Interview Questions
        ↓
Submit Answers
        ↓
AI Evaluation & Scoring
        ↓
Follow-up Questions & Feedback
```
